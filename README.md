# eKharid
Ecommerce website


## 🛠 Tech Stack, Libraries & Tools
**Frontend:**  
- React.js (Hooks & Context API)  
- Next.js (App Router)  
- Tailwind CSS  


**Backend:**  
- Node.js & Express.js  
- MongoDB with Mongoose ORM  
- JWT Authentication & bcrypt  
- Nodemailer  

**State Management & Forms:**  
- Redux Toolkit  
- Formik & Yup
